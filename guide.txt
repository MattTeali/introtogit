VCS (Version Control System)

basics commands

$ git init //Initialize Local Git Repository

$ git add <file> //Add File to Index (Staging)

$ git rm <file> //Remove File from Index (Staging)

$ git status //Check Status of Working Tree and Staging Area

$ git commit //Commit Changes of Staging Area

$ git push //Push local repo to remote repo

$ git pull //Pull changes from remote repo

$ git clone //clone repo into new folder

$ touch //Create new file

$ git config //Configure details

$ git add . //add all

$ git branch <name> //Makes new branch

$ git checkout <name> //Goes to other branch

$ git remote add origin https://github.com/MattTeali/introtogit.git

$ git push -u origin master
